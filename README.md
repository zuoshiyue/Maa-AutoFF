- 自动钓鱼
- 金蝶小游戏

## 使用说明

## 开发相关
### 技术栈
- 前端技术
  - javaScript
  - Vue 3
  - Pinia
  - Vite
  - SASS
  - Vue Router
  - Element Plus
- 后端技术
  - pywebview
  - opencv
  - numpy
  - rapidocr
- 打包
  - npm
  - Nuitka

### 运行方法
- `git clone https://github.com/Jai-wei/Maa-AutoFF.git`
- `pip install requirements.txt`
- `cd frontend && npm install`
- `npm run build`
- `python AutoFF.py`
